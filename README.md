# Javascript debug sample project

## requirement

- [Node.js 8.x](https://nodejs.org/)
- [Visual Studio Code](https://code.visualstudio.com/)

## install

```
$ npm install
```

## run

```
$ npm start
```

## e2e test

run selenium

```
$ npm run selenium
```

run e2etest

```
$ npm run e2e-test
```
